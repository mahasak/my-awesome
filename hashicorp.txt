#  Product for DevOps life from Hashicorp
>go lang and git is require to install with "go get"
> 




**Terraform**

`go get -u github.com/hashicorp/terraform`
> Terraform is a tool for creating, combining, and managing infrastructure resources across multiple providers. Manage resources on cloud providers such as Amazon Web Services, Google Cloud, Microsoft Azure, DNS records on DNSimple, Dyn, and CloudFlare, email services on Mailgun, and more.

**Serf**

`go get -u github.com/hashicorp/serf`

>Serf is a lightweight and decentralized solution for cluster membership, failure detection, and orchestration. Serf is used as a building block for highly available systems, such as Consul.

**Consul**

`go get -u github.com/hashicorp/consul`

>Consul is a distributed, highly-available, and multi-datacenter aware tool for service discovery, configuration, and orchestration. Consul enables rapid deployment, configuration, and maintenance of service-oriented architectures at massive scale.

**Vault**

`go get -u github.com/hashicorp/vault`

>Vault secures, stores, and controls access to tokens, passwords, certificates, API keys, and other sensitive resources in modern datacenters.For each resource, Vault handles leasing, revocation, rolling, and auditing.

**Nomad**

`go get -u github.com/hashicorp/nomad`
>Nomad is a cluster manager and scheduler that provides a common workflow to deploy applications across an infrastructure. Deploy virtualized, containerized, or standalone application workloads across a fleet of servers to maximize resource utilization.

**Otto**

`go get -u github.com/hashicorp/otto`
>Otto is a tool to develop and deploy any application on any cloud platform, all controlled with a single consistent workflow to maximize the productivity of you and your team.

**Govendor**

`go get -u github.com/kardianos/govendor`
>Go vendor tool that works with the standard vendor file.

**Packer**

`go get -u github.com/mitchellh/packer`
>Packer is a tool for creating images for platforms such as Amazon AWS, OpenStack, VMware, VirtualBox, Docker, and more &mdash; all from a single source configuration.
